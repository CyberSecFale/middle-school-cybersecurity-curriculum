# Cybersecurity & Digital Safety: Middle School Semester Proposal
A 16-week elective course designed to prepare students (ages 12-15) for future careers in tech, cybersecurity, and coding.

## 🚀 Course Overview
This curriculum bridges the gap between basic computer use and professional STEM readiness. Students transition from passive technology consumers to secure, logical, and ethical technology creators using industry-standard tools.

### 🎯 Key Learning Objectives
* **Technical Literacy:** Linux terminal navigation, networking fundamentals, and Python automation.
* **The Security Mindset:** Developing critical problem-solving and ethical "White Hat" hacking concepts.
* **The GitHub Advantage:** Building a 5-year career head start by documenting projects via Version Control.

## 📅 16-Week Curriculum Structure
| Weeks | Topic | Practical Labs |
| :--- | :--- | :--- |
| 1-2 | Digital Safety & GitHub | Setup secure GitHub portfolio & MIT Licensing. |
| 3-4 | Networking Fundamentals | Mapping the internet and IP communication. |
| 5-6 | Linux Command Line | Terminal navigation and file system management. |
| 7-8 | Intro to Cybersecurity | Defensive security & Cryptography basics. |
| 9-10| Python Programming | Variables, loops, and security scripting. |
| 11-12| Web Security Basics | Understanding vulnerabilities responsibly. |
| 13-14| Capture The Flag (CTF) | Solving beginner cybersecurity puzzles. |
| 15-16| Final Capstone Showcase | Publishing a full project portfolio to GitHub. |

## 🏗️ Portfolio & Documentation
This course emphasizes professional documentation. Students are required to:
1. **Maintain a GitHub Repository:** Every lab completion is documented with screenshots and structured `README.md` files.
2. **Open Source Participation:** Students learn to apply **MIT Licenses** to their projects, allowing for collaborative and ethical software sharing.
3. **Visual Proof:** Repositories include screenshots of successful *TryHackMe* and *OverTheWire* lab completions.
